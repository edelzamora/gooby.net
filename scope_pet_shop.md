## Scope

Pet shop with CRUD functionality. 3 parts being frontend, backend, and db.

### Tech

Frontend:

- Reactjs
- node

Backend:

- Flask or FastAPI

DB:

- PostgreSQL

HOSTING:

- AWS
- Netlify
- Heroku

Respositories:

- Git
- Github

## Deliverables

Front-end
Back-end (http points)
DB (backend connects/calls to db)

FRONTEND:

- Login
- Signup/Create Account
- Home/Menu/About page
- Pet/market to view
- user/settings page
- Navigation bar
- admin section to update or create pets objects

BACKEND:
Endpoints:

- login (POST)
- signup (POST)
- pets (GET)
- settings (PUT/PATCH)
- create (POST/PUT/PATCH)

DB:
Create tables, schema, relations, rows, columns
